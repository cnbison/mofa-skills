MoFA FM API

MoFA FM 播客平台 API，用于浏览节目、搜索内容、管理创作者内容、获取热搜数据。

Base URL: https://mofa.fm/api/

认证方式

  • 公开接口: 无需认证
  • 私有接口: JWT Bearer Token
Authorization: Bearer <access_token>

快速开始

1. 获取 Token

curl -X POST https://mofa.fm/api/auth/login/ \
  -H "Content-Type: application/json" \
  -d '{"username":"your_username","password":"your_password"}'

2. 调用 API

curl https://mofa.fm/api/podcasts/shows/ \
  -H "Authorization: Bearer <access_token>"

3. Python 示例

import requests

BASE_URL = "https://mofa.fm/api"

# 登录
token = requests.post(f"{BASE_URL}/auth/login/", json={
    "username": "user",
    "password": "pass"
}).json()["access"]

headers = {"Authorization": f"Bearer {token}"}

# 创建节目
show = requests.post(
    f"{BASE_URL}/podcasts/shows/create/",
    headers=headers,
    json={"title": "新节目", "content_type": "podcast"}
).json()

节目 API

GET /podcasts/shows/
获取节目列表（公开）
POST /podcasts/shows/create/
创建节目（需要认证）
{"title": "节目名称", "description": "...", "content_type": "podcast"}
GET /podcasts/shows/<slug>/
获取节目详情（公开）
PUT /podcasts/shows/<slug>/update/
更新节目（需要认证）
DELETE /podcasts/shows/<slug>/delete/
删除节目（需要认证）
获取节目单集: GET /podcasts/episodes/?show_slug=<slug>

单集 API

GET /podcasts/episodes/
获取单集列表（公开）
GET /podcasts/episodes/?show_slug=<slug>
获取指定节目的单集列表（公开）
GET /podcasts/episodes/<id>/
获取单集详情（需要认证）
POST /podcasts/episodes/create/
创建单集（需要认证，需要上传音频文件）
POST /podcasts/episodes/<id>/play/
增加播放计数（公开）

热搜 API

支持 56 个数据源: bilibili, zhihu, weibo, github, v2ex, douyin, 36kr, juejin, sspai, hackernews 等

GET /podcasts/trending/sources/
获取所有热搜数据源列表
GET /podcasts/trending/<source>/
获取指定数据源的热搜数据
// 示例: /podcasts/trending/bilibili/
{

搜索 API

GET /search/?q=<keyword>

GET /search/suggestions/?q=<prefix>

GET /search/popular/
热门搜索

互动 API

GET /interactions/episodes/<id>/comments/
获取评论列表（公开）
POST /interactions/comments/create/
发表评论（需要认证）
{"episode": 123, "text": "评论内容"}
POST /interactions/episodes/<id>/like/
点赞/取消点赞（需要认证）
POST /interactions/shows/<id>/follow/
关注/取消关注（需要认证）

脚本会话 API

AI 辅助创作播客脚本功能

GET /podcasts/script-sessions/
获取脚本会话列表（需要认证）
POST /podcasts/script-sessions/
创建脚本会话（需要认证）
{"title": "AI话题", "description": "讨论AI", "content_type": "podcast"}
POST /podcasts/script-sessions/<id>/chat/
AI 对话生成脚本（需要认证）
{"message": "帮我写一个关于AI的播客脚本"}
POST /podcasts/script-sessions/<id>/upload/
上传参考文件（需要认证）
POST /podcasts/script-sessions/<id>/finalize/
将脚本会话转为播客单集（需要认证）
{"show_slug": "my-show", "title": "AI播客第1期"}

使用示例

获取 B 站热搜

curl https://mofa.fm/api/podcasts/trending/bilibili/

搜索播客

curl "https://mofa.fm/api/search/?q=AI"

完整 Python 示例

from fm_client import MoFAFMClient

client = MoFAFMClient()

# 公开 API
shows = client.list_shows()
trending = client.get_trending("bilibili")
results = client.search("AI")

# 需要认证
client.login("username", "password")
me = client.me()
my_shows = client.my_shows()

# 脚本管理
session = client.create_script_session("AI话题讨论", "讨论AI最新进展")
response = client.chat_script_session(session["id"], "帮我写一个播客脚本")

错误处理

状态码 含义
200成功
201创建成功
400请求参数错误
401未认证
403无权限
404资源不存在
500服务器错误
注意: Token 有效期为 1 小时，refresh token 有效期为 7 天。
